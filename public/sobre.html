Sobre

O "Conecta Empresas" é um sistema projetado para a venda de pacotes de dados voltado para diferentes nichos de mercado, como imobiliário, automotivo e turismo, com foco na personalização conforme as necessidades de cada cliente.O sistema inclui funcionalidades essenciais, como o cadastro e login de usuários, garantindo uma experiência segura e fluida. Após o login, os usuários são redirecionados automaticamente para a página inicial, onde podem explorar os pacotes disponíveis. Uma das principais funcionalidades do sistema é a integração com a API PIX, permitindo que os usuários realizem pagamentos online através de QR Codes gerados dinamicamente, além da integração com o WhatsApp para confirmar transações.O design da interface foi cuidadosamente desenvolvido para ser responsivo, garantindo acessibilidade tanto em dispositivos móveis quanto em desktops, utilizando Bootstrap e outros frameworks modernos.
